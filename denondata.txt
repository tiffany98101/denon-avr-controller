== Offline static field catalog ==
Known read-only/exposed Denon AVR fields and endpoints supported by this tool.
This is not a list of hidden firmware internals.

Receiver identity
  receiver_name          type 3 XML / FriendlyName
  receiver_ip            discovered receiver IP

Network / HEOS
  heos_model             HEOS player/get_players / model
  heos_version           HEOS player/get_players / version
  network                HEOS player/get_players / network

Main Zone
  main_zone_name         type 6 XML / MainZone
  power                  type 4 XML / MainZone/Power
  source_index           type 7 XML / Zone zone=1 @index
  source_name            type 7 XML / Zone zone=1 source map lookup
  volume_raw             type 12 XML / MainZone/Volume
  volume_db              raw volume converted to dB
  volume_max_db          type 12 XML / MainZone/Max
  muted                  type 12 XML / MainZone/Mute

Zone 2
  zone2_name             type 6 XML / Zone2
  power                  type 4 XML / Zone2/Power
  source_index           type 7 XML / Zone zone=2 @index
  source_name            type 7 XML / Zone zone=2 source map lookup
  volume_raw             type 12 XML / Zone2/Volume
  volume_db              raw volume converted to dB
  muted                  type 12 XML / Zone2/Mute

Sources
  main_zone_sources      type 7 XML / Zone zone=1 source list
  zone2_sources          type 7 XML / Zone zone=2 source list

Audio / surround
  sound_mode             telnet MS? / SYSMI

Sleep timer
  main_zone_sleep        telnet SLP?
  zone2_sleep            telnet Z2SLP?

Tone / Audyssey
  dynamic_eq             telnet PSDYNEQ ?
  dynamic_volume         telnet PSDYNVOL ?
  cinema_eq              telnet PSCINEMA EQ. ?
  multeq                 telnet PSMULTEQ ?
  bass                   telnet PSBAS ?
  treble                 telnet PSTRE ?

Now Playing
  title                  formNetAudio_StatusXml.xml / Song
  artist                 formNetAudio_StatusXml.xml / Artist
  album                  formNetAudio_StatusXml.xml / Album

Raw XML endpoints
  type_3                 /ajax/globals/get_config?type=3
  type_4                 /ajax/globals/get_config?type=4
  type_6                 /ajax/globals/get_config?type=6
  type_7                 /ajax/globals/get_config?type=7
  type_12                /ajax/globals/get_config?type=12
  discovered_types       /ajax/globals/get_config?type=N for N=0..DENON_DATA_DISCOVERY_MAX_TYPE

Web UI information
  general_page           /general/general.html
  discovered_links       same-host read-only HTML/JS/XML/AJAX links referenced by the receiver UI

Discovered read-only endpoints
  safe_ajax_paths        read-only URL-looking strings found in receiver HTML/JS
